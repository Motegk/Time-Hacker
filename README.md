Time Hacker — это 2D платформер-головоломка на чистом HTML, CSS и JavaScript, где главная механика построена на перемотке времени. Игрок может откатывать последние секунды назад, создавать клонов, которые повторяют прошлые действия, и с их помощью проходить уровни с кнопками, дверями, лазерами, порталами, водой, гравитационными зонами, фазовыми стенами, магнитными рельсами и боссами.

В игре есть:
- одиночный и кооперативный режим;
- встроенный редактор уровней;
- магазин скинов и усилений;
- десятки уровней с разными стилями и механиками;
- паркур, временные пазлы и битвы с боссами.
